# Python-with-Turtle ‚Äî Versioned Scripts

This project organizes all Python Turtle scripts into multiple versions (`v1` through `v12`).

## Structure

- Each script from the original collection is available in twelve versions:
  - `v1_script.py`
  - `v2_script.py`
  - ...
  - `v12_script.py`

## How to Use

- Run the provided website (see `app.py`) to browse available versions and download scripts.
- Each script starts with a brief description (taken from its first comment or docstring).

## Website

Run the Flask app provided in `app.py`:

```bash
python app.py
```

Open [localhost:5000](http://localhost:5000) in your browser.
